## ארכיטקטורה

- **אין backend.** האתר סטטי; כל הנתונים נשמרים ב-`localStorage` של הדפדפן תחת המפתח `mln-finance-v1`, כאובייקט `db` אחד (JSON).
- **המשמעות:** הנתונים פר-מכשיר ופר-דפדפן. אין סנכרון בין טלפון למחשב — העברה ידנית דרך "גיבוי ושחזור" (העתקת טקסט JSON). סנכרון ענן משפחתי הוא הפיצ'ר המתוכנן הבא (Supabase או דומה).
- **פריסה:** כל push ל-`master` מתפרסם אוטומטית ב-Pages תוך דקה-שתיים.

## מבנה הנתונים (`db`)

| שדה | תוכן |
|---|---|
| `transactions` | תנועות: `{id, kind: income/expense, category, amount, note, date, recurringId?, eventId?}` |
| `investments` | אפיקי השקעה: `{id, name, kind, value}` — `name` ריק = מציגים את שם הסוג; הסדר במערך הוא סדר התצוגה (נגרר) |
| `recurring` | הוצאות קבועות; נרשמות אוטומטית לכל חודש (`postRecurring`), מחיקה מוסיפה ל-`skip` |
| `projects` | פרויקטים לחיוב; סטטוס `paid` יוצר תנועת הכנסה מקושרת (`linkedTxn`) |
| `events` | אירועים (טיול וכו') עם פריטי הוצאה; "העברה לסיכום" יוצרת הוצאה מקושרת אחת |
| `budgets` | תקציב חודשי לקטגוריה |
| `osh` | עו"ש: `{base, month}` — base = יתרת בסיס לחודש; זמין = base + הכנסות − הוצאות (מודל "ארנק") |
| `snapshots` | שווי השקעות פר-חודש (לגרף שווי נקי) |
| `goal`, `navOrder`, `catOrder`, `activeMonth` | יעד חיסכון, סדר לשוניות, סדר קטגוריות, חודש פעיל (null = אוטומטי) |

## פיצ'רים עיקריים

- קלט חכם בעברית: "קניתי במכולת 60 שקל" → הוצאה/מזון/60 (`parseTransaction`, `parseAmount` כולל מספרים במילים)
- הכתבה קולית (Web Speech API, `he-IL`) עם נפילה למקלדת
- מסכים: סקירה, השקעות, תזרים, קבועות, פרויקטים, נתונים (+ אירועים)
- גרירת סדר: קטגוריות בתזרים ואפיקי השקעות (ידית ☰, מנוע `listDrag*`); סדר לשוניות דרך נתונים → הגדרות
- תקציב פר-קטגוריה + התראות חריגה ויתרה נמוכה
- סטטיסטיקה: שווי נקי 6 חודשים, הכנסות/הוצאות, פילוח, KPIs
- גיבוי/שחזור בהעתקת טקסט, ייצוא CSV

## כללי עבודה (חובה!)

1. **על כל שינוי** מעדכנים שלושה מקומות יחד: `APP_VERSION` ב-index.html, `version` ב-package.json, ורשומה ב-CHANGELOG.md.
2. מספר הגרסה מוצג באפליקציה בפינה השמאלית העליונה — כך מוודאים שהעדכון הגיע למכשיר.
3. commit + push אחרי כל שינוי; לוודא שה-Pages התעדכן (grep על `APP_VERSION` ב-HTML החי).
4. טקסט משתמש ל-innerHTML — תמיד דרך `esc()`.
5. סכומים מהקלט — תמיד דרך `parseNum` (תומך מינוס).
